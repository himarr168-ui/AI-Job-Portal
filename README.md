# AI Job Portal

An AI-powered Job Portal built using Python. This project automates resume parsing, ATS scoring, candidate screening, interview assistance, and job matching using Artificial Intelligence.

---

## Project Structure

```
AI-Job-Portal/
│
├── ats_engine/ # ATS scoring and resume evaluation
├── data/ # Dataset and sample resumes
├── interview_ai/ # AI interview question generation
├── logs/ # Application log files
├── parsers/ # Resume parsing modules
├── scoring/ # Candidate scoring algorithms
├── screening_ai/ # AI-based candidate screening
├── tests/ # Unit tests
├── utils/ # Helper utilities (logger, helpers)
│
├── main.py # Main application entry point
├── requirements.txt # Python dependencies
├── README.md # Project documentation
└── .gitignore # Git ignore rules
```

## Installation

Clone the repository:

```bash
git clone https://github.com/himarr168-ui/AI-Job-Portal.git
```

Go to the project folder:

```bash
cd AI-Job-Portal
```

Create virtual environment:

```bash
python -m venv venv
```

Activate the virtual environment:

Windows:

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python main.py
```
