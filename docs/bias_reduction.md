# Bias Reduction Documentation

## Objective
To improve fairness and reduce bias during resume evaluation.

## Implemented Improvements
- Resume text normalization
- Personal information masking
- Score normalization
- Fair candidate evaluation
- Bias risk assessment

## Masked Attributes
- Name
- Gender
- Age
- Date of Birth
- Religion
- Nationality
- Marital Status
- Address

## Result
The ATS evaluates candidates based on skills, education, experience, and semantic similarity rather than personal attributes, ensuring a fair and unbiased recruitment process.
