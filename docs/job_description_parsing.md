# Job Description Parser

## Features
- Reads Job Description from .txt file
- Cleans unwanted spaces
- Extracts:
  - Role
  - Skills
  - Experience
  - Education

## Output

{
    "role": "Python Developer",
    "skills": ["Python", "Django", "SQL", "Git", "REST API"],
    "experience": "2+ years",
    "education": "B.Tech / BCA / MCA"
}
